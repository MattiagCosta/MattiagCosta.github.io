Whole Numbers

Whole Numbers are strings of figures, defined by the user, that represent a positive number. You can compare and do operations with Whole Numbers, as well as other things. The user can work with Whole Numbers not only in the decimal base, but in any base they can think of. The concept is flexible, so that the source code for Whole Numbers can be written in multiple programming languages.

Download for C++

Download the necessary files to work with Whole Numbers: WholeNumbers.cpp and WholeNumbers.h.

C tools

Dynamic List

A Dynamic array that stores pointers to the heap. This library offers functions to work with this type of list, such as adding and searching elements. Storing data in the heap is a common task in C-programming: the Dynamic List can help achieving this task easily. Dynamic Lists are allocated in the heap too, so they need to be freed with the functions provided by the library.
Download the necessary files here: DynamicList.c, DynamicList.h.

Lapse Counter

This weirdly named structure counts how many times it is called within a certain amount of time. It can be useful to count the iterations of a loop every second, or to use it as a timer.
Download these files to use Lapse Counters: LapseCounter.c, LapseCounter.h.
